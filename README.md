# PokéMartin

Boutique Pokémon en ligne + outil de gestion admin.

## Fichiers
- `index.html` — **Boutique client** (public, accessible sur pokemartin.com)
- `pokemartin-admin.html` — **Outil admin** (privé, connexion requise)

## Connexion admin
Créez un utilisateur dans Firebase Console → Authentication → Add user
(Email/Password)

## Déploiement
Ce site est hébergé via GitHub Pages.
